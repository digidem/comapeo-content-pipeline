## **CoMapeo Mobile at First Glance**

The primary view of CoMapeo is the ![app-icon-map](assets/c3a90bf1bbc7f33b79ebc8060e2591c37c754808d7c30e3f9b9c27848ae5c11a.png) Map view with access to mapping features and the main menu. This is serves as the home screen when restarting CoMapeo onboarding is complete. 

The map controls include ![app-icon-comapeo-map-recenter](assets/27e12ea7ab4f14260ce1930f689682d0a5cefa9e6730dd97899c0ae3cd349af6.jpg)recenter, ![app-icon-comapeo-map-pan](assets/de3946578345ea1b7d58fc0f9efda14565ecfaf2c52061e918fa28e54b400589.jpg) slide-to-pan, and pinch-to-zoom with a dynamic scale. Real time GPS sensor information can be viewed in ![app-icon-gps20](assets/9dba27a9d6b5217674fc683388d592f4a825c79bcfee67c8732ac231d915aa3d.png) GPS Details.

## CoMapeo Settings

### ![app-icon-comapeo-device-name](assets/d1f67acffc237e7d29830756e6b11cfcdab7f0188883efba3a64946062eb0a4c.png)** Device Name**

The device is named as part of the onboarding process. Device names are displayed in CoMapeo collaboration features like inviting teammates to join a project and viewing teammates in a project. Device names cannot be changed by any other device.  The only way to change a device name for use in CoMapeo is to edit it from this menu option on that same device. 

### ![app-icon-language](assets/50ab50695da3c567ca05396a5df8219510128a3de5c4eb757dc622921964618c.png)** Language**

Language options available may include languages with anywhere between 1% and 100% translations. 

### ![app-icon-coordinate-system](assets/7c6073e70a3daa2ffbb4cbfb883b8b7ce3ec23ffc50a13fedcc3dca1dc71e2d8.png) **Coordinate System**

Choose one of the following options:

:::note 💡 Tip
Select the ![app-icon-coordinate-system](assets/7c6073e70a3daa2ffbb4cbfb883b8b7ce3ec23ffc50a13fedcc3dca1dc71e2d8.png) Coordinate System option that is used most commonly in your context.
:::

### ![app-icon-data-privacy](assets/0b8b4afb48cdbb5c301a419f338e42620bd7caeee60036ebf95311959f1732c8.png) **Data & Privacy**

Choose to **opt in** or **opt out** of sharing diagnostic data.

:::note 👉🏽
Go to 🔗 [Adjusting Data Sharing & Privacy](/docs/adjusting-data-sharing-and-privacy) to learn more.
:::

### ![app-icon-about-info](assets/fe1ffc013797302646ad516b9e461e490c7cb704b53dbf76df30e74a02923aa2.png) **About CoMapeo**

View app version  number and more details that may help with diagnostics when accessing technical support from the CoMapeo Help team.

### ![app-icon-comapeo-security](assets/24aee61dd87ccbf54ff80166d4429131e1a41250bdab6d1a484330dacdbcff5e.png) **Security**

Security tools to aid with physical security threats.

:::note 👉🏽
Go to 🔗 [Using an App Passcode for Security](/docs/using-an-app-passcode-for-security) to learn more.
:::
