# Comenzando

Esta guía te ayudará a instalar y configurar la aplicación en tu dispositivo. Sigue los pasos a continuación para comenzar rápidamente.

## Requisitos del Sistema

- Sistema Operativo: Windows 10+, macOS 10.15+, o Linux

- RAM: Mínimo 4GB, recomendado 8GB

- Almacenamiento: 500MB de espacio en disco disponible

## Pasos de Instalación

1. Descarga el instalador desde el sitio web oficial

1. Ejecute el instalador y siga el asistente de configuración

1. Inicie la aplicación después de que se complete la instalación.

:::note 💡
Asegúrate de reiniciar tu dispositivo después de la instalación para que todos los cambios surtan efecto.
:::
