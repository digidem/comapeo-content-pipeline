# Alterando Conjunto de Categorias

## O que é necessário para alterar um conjunto de categorias?

✔️ Arquivo de conjunto de categorias personalizado: .comapeocat

✔️  As ferramentas do ![app-icon-comapeo-coordinator](assets/77ddbc4137798bed6b6b838cdfc22c990548dd8dc643056e500ffb59b0eb2b1b.png) Coordenador estão visíveis.

✔️  O dispositivo possui uma função de  ![app-icon-comapeo-coordinator](assets/77ddbc4137798bed6b6b838cdfc22c990548dd8dc643056e500ffb59b0eb2b1b.png) Coordenador.

### Acessando as ferramentas do Coordenador

Ao fazer o mapeamento por conta própria, as ferramentas do coordenador são exibidas após seguir as etapas para **iniciar um novo projeto**.

Acesse 🔗 [Criando um novo projeto](https://www.notion.so/docs/creating-a-new-project)

### Carregando um novo arquivo de conjunto de categorias

O tipo de arquivo **.comapeocat **é propriedade do CoMapeo. Não pode ser aberto por nenhum outro aplicativo e nem pode ser aberto pelo Android. Use o CoMapeo para carregar o arquivo de conjunto de categorias no projeto desejado. Essa função é permitida somente àqueles que são coordenadores de projetos.

:::note 👣
### **Passo a Passo: Mobile**

***Passo 1: ***Procure o arquivo na sua pasta de downloads e transfira-o para o dispositivo móvel em que você está usando o CoMapeo. Você pode usar um e-mail, Google Drive, WhatsApp ou a ferramenta que funcionar melhor para você para obter arquivos no seu telefone.

:::note 🖼️
static/images/changingcategoriesse_0.png
:::

***Passo 2: ***Abra o ![comapeo-platform](assets/e67891bb2deee359f763589bf2283850bff9acd6208a5cc1e90171070d897de3.png) CoMapeo e, no menu principal, **selecione o projeto **onde o novo conjunto de categorias é necessário.

***Passo 3: ***Abra as **Ferramentas do Coordenador.**

:::note ⚠️ Aviso
Isso não é visível para os Participantes de um projeto
:::

***Passo 4: ***Selecione **Categorias de Projetos**.

:::note 🖼️
static/images/changingcategoriesse_1.png
:::

***Passo 5: ***Toque em **Importar Categorias **para abrir o seletor de arquivos do seu sistema operacional

:::note 🖼️
static/images/changingcategoriesse_2.png
:::

***Passo 6: ***Encontre o arquivo de conjunto de categorias (.comapeocat) e selecione-o. O CoMapeo tentará importar o arquivo selecionado. Se for bem-sucedido, as informações na tela de Categorias serão atualizadas.

<div class="notion-spacer" aria-hidden="true" role="presentation"></div>

---
:::

:::note 💡 Dica
Os conjuntos de categorias estão incluídos nos dados do projeto e são compartilhados com novos colegas de equipe, após a conclusão do convite para projetos. Eles também são incluídos ao fazer Trocas com colegas de equipe para ajudar a manter tudo atualizado.
Acesse: 🔗[Entendendo Como a Troca Funciona](https://notion.so/docs/understanding-how-exchange-works)** **para explicação completa
:::

## Revisando um Conjunto de Categorias

Para revisar o Conjunto de Categorias incluído no CoMapeo dentro do próprio aplicativo, siga os passos para [criar uma observação, ](https://notion.so/docs/creating-a-new-observation)[adicionar detalhes](https://notion.so/docs/creating-a-new-observation#adding-details) e exclua as observações salvas como parte da exploração das categorias.

Se esta for a primeira vez configurando um projeto ou alterando o conjunto de categorias, é provável que seu próximo passo seja convidar colaboradores para coletar dados como uma equipe.

Acesse: 🔗[Convidando Colaboradores](https://notion.so/docs/inviting-collaborators)** **para instruções
