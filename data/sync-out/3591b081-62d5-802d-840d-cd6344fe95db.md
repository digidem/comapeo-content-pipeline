:::note ⚠️ Warning
Internet-connected sharing only works with ![app-icon-comapeo-remote-archiver](assets/70c2034a3fd6001eecfa493c0a164118f17e6c67ef4928659332f939ee2215e0.png) Remote Archive enabled.
:::

To exchange goods or services using an internet connection, five requirements must be met:

1. Two or more devices with CoMapeo ![comapeo_logo_circle](assets/3a40065bd081188fd54f3a2eea142a94ef06d828f89e0980a51391cb7aa347c9.png)

1. The devices must have internet access.

1. Preparation of a ![app-icon-comapeo-remote-archiver](assets/70c2034a3fd6001eecfa493c0a164118f17e6c67ef4928659332f939ee2215e0.png) **Remote Archive** is completed.

1. The devices must belong to the same project where ![app-icon-comapeo-remote-archiver](assets/70c2034a3fd6001eecfa493c0a164118f17e6c67ef4928659332f939ee2215e0.png) **Remote Archiving** is enabled.

1. New observations or routes to exchange.

![image](assets/67fea0702faed518564f0b8ed7417a86fe4b1dc97474aba789190dd03a65d518.png)

:::note 🔵 Visit 🔗** **[How the Exchange Works](https
//www.notion.so/docs/understand-how-the-exchange-works) for a full description and more details.
:::

:::note 👣
### Step by Step

***Step 1: ****Open the* ![three-line-menu-black](assets/351bd29be717f58fba4061f4d5ef0db5508de31bd77417f31c2248b9f6cbbdee.jpg)** Menu**

---

***Step 2: ****Select**** *** ![app-icon-comapeo-remote-archiver](assets/70c2034a3fd6001eecfa493c0a164118f17e6c67ef4928659332f939ee2215e0.png) **Intercambiar**  

***Step 3:*** The screen will show which devices are connected.

When a project has ![app-icon-comapeo-remote-archiver](assets/70c2034a3fd6001eecfa493c0a164118f17e6c67ef4928659332f939ee2215e0.png) **Remote Archiving **enabled, it will appear connected whenever there is an internet connection. This will be displayed just below the large ![app-icon-comapeo-exchange](assets/d1f75c78a51b8e5a813245aa74758646c26969350099762a8dc34b327ee517c7.png)**Exchange** icon.

![image](assets/9cb6fb978bc6874c5f20889dc64fc20ad2ed53a0265fa6b17d22d80992c30b87.png)

:::note 👉🏽 More
Exchange with ![app-icon-comapeo-remote-archiver](assets/70c2034a3fd6001eecfa493c0a164118f17e6c67ef4928659332f939ee2215e0.png) **Remote Archive** works the same whether or not there are other devices connected via WiFi.
:::

![image](assets/91b9d3fc85f81112b9edb798b527f6e9d40ecbb0f05ccaf96005ef29e6df5a8e.png)

:::note 💡 Tip
**Adjust the Exchange settings if necessary.** Swap settings can be modified to optimize device performance.
Go to 🔗 [Understand How Exchange Works →  Exchange Settings ](https://www.notion.so/digidem/docs/understand-how-swap-works#swap-settings)
:::

---

***Step 4: ***Tap** Start** to begin the exchange.

---

***Step 5: ***It will display **Complete** when all observations have been exchanged. Tap** Done** to return to the ![three-line-menu-black](assets/351bd29be717f58fba4061f4d5ef0db5508de31bd77417f31c2248b9f6cbbdee.jpg)**Menu**
:::
