## Integração

:::note 🖼️
static/images/installingcomapeoonb_6.png
:::

<div class="notion-spacer" aria-hidden="true" role="presentation"></div>

Esta é uma etapa necessária para ajudar a configurar dispositivos, permitindo que o CoMapeo use recursos disponíveis, incluindo armazenamento e sensores essenciais como câmera e GPS. Para manter essa experiência fácil, o CoMapeo solicitará permissão para acesso a sensores adicionais na primeira vez que usar recursos específicos, como trilhas e áudio. Incluídos na integração estão:

**Tela de Boas-Vindas
**Você será convidado a [Começar] e solicitado permissões de dispositivo ao iniciar.

**Política de Dados e Privacidade
**O aplicativo apresentará os dados que coleta, e você pode escolher [optar por participar] ou [optar por não participar]. Isso também pode ser alterado posteriormente via

**Configurações > Privacidade de Dados
**Veja a Política de Privacidade em inglês: [CoMapeo Dados e Privacidade](https://digidem.notion.site/CoMapeo-Data-Privacy-d8f413bbbf374a2092655b89b9ceb2b0)

**Nomeação de Dispositivos
**Você deve inserir um nome para o dispositivo. Este é o nome que outros dispositivos verão ao convidá-lo para um projeto e será exibido na lista de membros do projeto.

<div class="notion-spacer" aria-hidden="true" role="presentation"></div>

### **Passo a Passo: Mobile**

---

***Passo 1: *****Conceder permissão para o CoMapeo tirar fotos.
**A câmera do CoMapeo é um dos principais recursos do aplicativo. Sem essa permissão, a câmera exibirá apenas uma tela preta. Outras partes do aplicativo continuarão funcionando.

:::note 🖼️
static/images/installingcomapeoonb_7.jpg
:::

---

***Passo 2: *****Conceder permissão para acessar localização**

O CoMapeo precisa acessar a localização de um dispositivo para coletar coordenadas GPS para salvar em Observações. Sem essa permissão, o aplicativo ainda poderá ser usado em um grau limitado. Observações ainda podem ser salvas com uma opção para inserir coordenadas GPS manualmente.

Versões mais recentes do Android oferecem uma opção de dados de localização precisos ou aproximados. **Escolha com precisão **para os resultados de coleta de dados mais precisos.

:::note 🖼️
static/images/installingcomapeoonb_8.jpg
:::

---

***Passo 3: ***Selecionar **Começar **na tela de boas-vindas do CoMapeo

:::note 🖼️
static/images/installingcomapeoonb_9.jpg
:::

<div class="notion-spacer" aria-hidden="true" role="presentation"></div>

---

***Passo 4: *****Saiba mais sobre Dados e Privacidade**

Saiba mais sobre como a Awana Digital construiu o CoMapeo para manter seus dados seguros e protegidos.

Clique em Saiba Mais para obter mais detalhes sobre privacidade de dados e quais dados são enviados para a Awana Digital para nos ajudar a melhorar o aplicativo.

Aqui você também pode escolher para **optar **ou **optar por não participar **de compartilhamento de dados de diagnóstico. Isso também pode ser alterado posteriormente via ![app-icon-app-settings](assets/2c4335fb69ab490d45c00fa57a40665ffeb87869bd446367c970909f95287356.png)Configurações → Privacidade de Dados.

Vá para 🔗[CoMapeo Dados e Privacidade](https://digidem.notion.site/CoMapeo-Data-Privacy-d8f413bbbf374a2092655b89b9ceb2b0)para revisar a Política de Privacidade em inglês

:::note 🖼️
static/images/installingcomapeoonb_10.jpg
:::

---

***Passo 5*****: Nomeie seu dispositivo**

:::note 🖼️
static/images/installingcomapeoonb_11.jpg
:::

- Nomes são necessários para todos os dispositivos no CoMapeo.

- Os nomes dos dispositivos são exibidos nos recursos de colaboração do CoMapeo, como convidar colegas de equipe para participar de um projeto e visualizar colegas de equipe em um projeto.

- Os nomes podem ser alterados com facilidade.

- A nomeação de dispositivos em um projeto de equipe pode precisar seguir protocolos ou acordos feitos com as pessoas da equipe.

- As opções para nomes incluem o nome da pessoa que possui o dispositivo, um apelido para o dispositivo ou um codinome usado em sua equipe por motivos de segurança.

:::note 💡 Dica
Use algo anônimo para contextos onde questões de segurança são uma preocupação. Cole um adesivo em cada telefone com seu nome se eles estiverem sendo usados por várias pessoas.
:::

---

***Passo 6: *****O CoMapeo está pronto para uso! **Escolha ![app-icon-comapeo-participant](assets/2dc5f8d1ce7cb4fc3056d93376d473bed776c782a82ce25d64e4cf658571a316.png) **Junte-se a um Projeto **ou![app-icon-comapeo-coordinator](assets/77ddbc4137798bed6b6b838cdfc22c990548dd8dc643056e500ffb59b0eb2b1b.png)**Mapear Sozinho**

Para explorar o CoMapeo e configurar um projeto, selecione ![app-icon-comapeo-coordinator](assets/77ddbc4137798bed6b6b838cdfc22c990548dd8dc643056e500ffb59b0eb2b1b.png)**Mapear sozinho**.

:::note 💡 Dica
Para ingressar em um projeto, um usuário diferente do CoMapeo deve ter um projeto configurado e deve convidar dispositivos para criar uma equipe no CoMapeo.
Vá para 🔗[Convidando Colaboradores](https://notion.so/docs/inviting-collaborators)para instruções
