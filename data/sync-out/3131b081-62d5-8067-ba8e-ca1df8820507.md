## Sobre Categorias no CoMapeo

### **O que é uma categoria no CoMapeo?**

Uma categoria é um ícone, rótulo e perguntas detalhadas predeterminadas para qualquer tipo de dado que está sendo coletado. Para cada observação feita com o CoMapeo, os participantes selecionam uma categoria para classificá-la e podem responder a perguntas ou a prompts para fornecer mais detalhes sobre o que está sendo documentado.

**O que são Conjuntos de Categorias no CoMapeo?**

Os Conjuntos de Categorias são arquivos que contêm e definem as diferentes Categorias de um Projeto. Criar configurações costumava exigir habilidades de programação que limitavam a possibilidade de as pessoas personalizarem suas próprias categorias. Agora tornamos isso mais acessível por meio de um modelo de planilha que explicamos nesta seção.

:::note 💡
Para aqueles familiarizados com o aplicativo antecessor Mapeo, os Conjuntos de Categorias eram anteriormente conhecidos como *Configurações.*
:::

## Por que personalizar Conjuntos de Categorias?

[VÍDEO PASSO A PASSO]

**Razões para personalizar Conjuntos de Categorias do CoMapeo**

- As Categorias Personalizadas são uma ferramenta poderosa para personalização no CoMapeo, permitindo que os usuários definam ícones e questionários específicos para seus projetos.

- O texto pode ser escrito em qualquer idioma que possa ser digitado, garantindo que as partes principais da interface de coleta de dados possam aparecer no idioma nativo dos grupos que a utilizam.

**Reflita e organize as informações que você precisará**

O planejamento inicial e a consulta com os participantes do projeto são essenciais para criar formas eficazes de organizar seus dados. Para iniciar o processo de personalização, os participantes do projeto devem refletir sobre os objetivos do projeto, que tipo de dados serão necessários e quais propriedades podem ser necessárias para as saídas de dados.

Você precisará pensar sobre o que incluir para cada uma das **principais áreas personalizáveis do CoMapeo.**

| Componente de Categorias | Descrição | Formato |
| --- | --- | --- |
| **Nome da Categoria** | Ao coletar ou criar dados com o CoMapeo, os usuários atribuem uma categoria de nível superior a cada observação ou elemento no mapa. | texto (recomendado menos de x caracteres para melhor exibição |
| **Ícone** | Cada Categoria que você incluir deve ter um ícone ou um pequeno gráfico para exibir aos usuários ao selecionar uma categoria. | png, (máximo 100px) |
| **Cor** | A cor aparece na visualização do mapa  e na Tela de seleção de categoria  Ela ajuda a definir grupos temáticos ao escolher cores que funcionam como uma legenda do mapa. | [Código de cor hexadecimal](https://htmlcolorcodes.com/color-picker/) |
| **Campos de detalhes** | Para cada Categoria no seu Conjunto de Categorias, você pode incluir um ou mais campos de dados estruturados (como um mini formulário ou questionário) que os usuários podem preencher opcionalmente ao criar uma nova observação ou elemento de mapa com essa categoria. | Prompt de texto com campos que podem ser preenchidos com valores que são texto digitado, seleção única ou seleção múltipla. Os valores podem ser texto. Observe que opções de seleção única podem ser preferidas para processamento de dados ao organizar ou fazer análises. |

## Usando o **CoMapeo Category Generator Spreadsheet**

O Gerador de Categorias CoMapeo é uma ferramenta em desenvolvimento ativo, que se destina a usuários avançados de tecnologia ou, no mínimo, a alguém muito confortável com o uso de planilhas disposto a aprender sobre o uso de plug-ins do Google Sheets.

Espera-se que aprendamos sobre problemas técnicos que você encontre ao usá-lo para nos ajudar a melhorá-lo.

Esta ferramenta permitirá que você crie e edite Categorias, ícones, campos de detalhes e idioma, e exporte todas essas informações para um arquivo do tipo **.comapeocat**

Tudo isso é feito preenchendo um conjunto de planilhas com as informações necessárias.

<div class="notion-spacer" aria-hidden="true" role="presentation"></div>

Acesse 🔗 [https://lab.digital-democracy.org/comapeo-category-spreadsheet-plugin/](https://lab.digital-democracy.org/comapeo-category-spreadsheet-plugin/) para instruções completas

## **Testando novas categorias**

Uma parte importante do processo é testar as categorias personalizadas criadas no CoMapeo Mobile. Pode ser útil incluir alguns colegas de equipe para ajudar a identificar quaisquer melhorias necessárias.

Acesse 🔗[Alterando Conjunto de Categorias](https://notion.so/docs/changing-categories-set)** **para instruções sobre como importar arquivos .**comapeocat**

As revisões devem ser feitas no **CoMapeo Category Generator. **Repita as etapas de exportação novamente e teste conforme necessário.
