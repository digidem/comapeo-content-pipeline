# Selecionando Funções de Dispositivo & Equipes

## O que é o “Mapeamento por conta própria”?

Quando o CoMapeo é instalado pela primeira vez em um novo dispositivo, ele é configurado para começar a coletar dados dentro de um projeto do qual é o único membro, não colaborando com nenhum outro dispositivo ou equipe. Isso pode ser ideal para indivíduos trabalhando de forma independente, sem necessidade de categorias personalizadas.

:::note 👉🏽 Nota
Alterar conjuntos de categorias requer ![app-icon-comapeo-coordinator](assets/77ddbc4137798bed6b6b838cdfc22c990548dd8dc643056e500ffb59b0eb2b1b.png) Ferramentas do Coordenador que se tornam visíveis após a configuração de um projeto. Não há necessidade de convidar dispositivos para um projeto ter Conjuntos de Categorias Personalizados.
Acesse 🔗[Alterando Conjunto de Categorias](https://notion.so/docs/changing-categories-set)** **para seguir os passos necessários
:::

Em alguns casos, as pessoas podem querer fazer backup e visualizar as informações que coletaram em um computador. Para fazer isso, instale o CoMapeo Desktop no computador e convide-o para o mesmo projeto. Este fluxo de trabalho funciona da mesma forma que convidar um celular diferente para um projeto - ambos exigem que o dispositivo original os convide para colaborar.

## Coletando Dados com uma Equipe e um Projeto

Um dos pontos fortes do CoMapeo é permitir que os usuários não apenas coletem dados por conta própria, mas também colaborem com outros como parte de uma equipe. Para isso, os usuários configuram novos projetos, dos quais se tornam Coordenadores, e então podem convidar outros membros como Coordenadores ou Participantes. Esses dois papéis têm permissões diferentes, descritas abaixo. Todos os usuários dentro do mesmo projeto podem trocar dados do projeto entre si e terão acesso ao mesmo conjunto de categorias para coletar dados. Eles também podem trocar dados com um Arquivo Remoto, se ele tiver sido configurado.

**O que é uma Equipe no CoMapeo**

![app-icon-comapeo-participant](assets/2dc5f8d1ce7cb4fc3056d93376d473bed776c782a82ce25d64e4cf658571a316.png) **Equipe **é a lista de dispositivos incluídos em um projeto e os papéis que cada dispositivo recebeu. As equipes podem incluir qualquer coisa a partir de dois dispositivos em diante. Todo dispositivo no projeto pode visualizar as informações da equipe para saber quais outros dispositivos fazem parte do seu projeto, e todo dispositivo pode visualizar todas as observações e trilhas que foram compartilhadas.

## Funções disponíveis no CoMapeo

![app-icon-comapeo-participant](assets/2dc5f8d1ce7cb4fc3056d93376d473bed776c782a82ce25d64e4cf658571a316.png) **Participantes **podem![app-icon-comapeo-add-observation](assets/a29cca65239d48f05dd3abf4db7c83f36eb65cd483866c560554e9638d528329.png) criar ![app-icon-coordinates](assets/16ad368673fa5ccbab0930b79526b9168194d8b4af5d94145d2d0d1088a5100b.png)Observações, registrar ![app-icon-comapeo-track](assets/bb93ade6fdc62318289db8417ed5eba91019281b6703339d0288ce4095006117.png)Trilhas,  ![app-icon-edit](assets/1e513871ec012d90a9f11451bb7284263ceb07cbf1ed7d13991db190473acf23.png) **Editar **suas próprias observações e distribuir informações usando ![app-icon-comapeo-exchange](assets/d1f75c78a51b8e5a813245aa74758646c26969350099762a8dc34b327ee517c7.png) **Trocar ,** ![app-icon-share](assets/a419d3830b128d8df965e5057ab5ad45c01336f9ea2bead41c8ac4a2971ceef4.png) **Compartilhar **ou **Baixar Observações.**

![app-icon-comapeo-coordinator](assets/77ddbc4137798bed6b6b838cdfc22c990548dd8dc643056e500ffb59b0eb2b1b.png)**Coordenadores **podem fazer tudo o que foi mencionado acima. Além disso, eles podem ![app-icon-edit](assets/1e513871ec012d90a9f11451bb7284263ceb07cbf1ed7d13991db190473acf23.png)Editar observações coletadas por outros e usar as ![app-icon-comapeo-coordinator](assets/77ddbc4137798bed6b6b838cdfc22c990548dd8dc643056e500ffb59b0eb2b1b.png) **Ferramentas do Coordenador **para convidar colaboradores, adicionar categorias personalizadas e habilitar recursos como o Arquivo Remoto.

As funções são atribuídas aos dispositivos pelo coordenador que os convida, durante o processo de convite. As funções são visíveis no menu e para todos os membros do projeto na tela da equipe.

:::note 🖼️
static/images/selectingdeviceroles_1.jpg
:::

:::note 👉🏽 Observação
Se uma função foi atribuída incorretamente, cancele o convite, se possível. Se o convite foi aceito, siga as etapas para remover o dispositivo. Em seguida, repita o processo de convite do dispositivo novamente, tomando cuidado para selecionar a função correta.
Veja 🔗[Remover um dispositivo de um Projeto](https://notion.so/docs/removing-a-device-from-a-project)
:::
