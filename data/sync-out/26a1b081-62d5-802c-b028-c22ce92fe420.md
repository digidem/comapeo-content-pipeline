## Starting a New Project

A new Project, will start with no observations, the CoMapeo Categories, and default Background Map. 

:::note 👣
### Step by Step

***Step 1: ***Go to the ![three-line-menu-black](assets/351bd29be717f58fba4061f4d5ef0db5508de31bd77417f31c2248b9f6cbbdee.jpg)**Menu**.

***Step 2:*** Tap on ![app-icon-comapeo-participant](assets/2dc5f8d1ce7cb4fc3056d93376d473bed776c782a82ce25d64e4cf658571a316.png) **Collaborate**.

***Step 3:*** Select ![app-icon-comapeo-coordinator](assets/77ddbc4137798bed6b6b838cdfc22c990548dd8dc643056e500ffb59b0eb2b1b.png) **Start New Project.**

***Step 4:*** Add a **project name** and continue

***Step 5:*** Select a ![-app-icon-comapeo-project-stats](assets/62edee383fbea1a2a7a9491250fb416621311f73aebf97c893acc4618da3263c.jpg) **Project Statistics **option. This can be changed in Coordinator Tools

***Step 5:*** Your new project is ready! 
:::

![image](assets/f0d994c90fe2da23b55c303f59b58be848be1a4359b94831f6635610037e935d.jpg)

:::note 💡 Tip
The project that you started when first onboarding onto CoMapeo will still be available go back to
:::

The device that creates the New Project is the Coordinator and has access to new menu options. ![app-icon-comapeo-participant](assets/2dc5f8d1ce7cb4fc3056d93376d473bed776c782a82ce25d64e4cf658571a316.png) Teams and ![app-icon-comapeo-coordinator](assets/77ddbc4137798bed6b6b838cdfc22c990548dd8dc643056e500ffb59b0eb2b1b.png) Coordinator Tools will appear in the ![three-line-menu-black](assets/351bd29be717f58fba4061f4d5ef0db5508de31bd77417f31c2248b9f6cbbdee.jpg)Menu

Go to 🔗 [Inviting Collaborators](/docs/inviting-collaborators) for instructions on inviting new devices to the team
