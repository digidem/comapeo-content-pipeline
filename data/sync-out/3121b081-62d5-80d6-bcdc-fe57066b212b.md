# Reunindo o Equipamento Adequado para o CoMapeo

## Por que é importante reunir o equipamento adequado para o CoMapeo?

O CoMapeo Mobile foi projetado para funcionar em uma variedade de dispositivos Android de diversos modelos. Ele utiliza o hardware básico que está incluído na maioria dos smartphones atualmente: uma câmera, GPS, bateria e memória. No entanto, se você planeja usar o CoMapeo em longas jornadas, ambientes offline e diversas condições climáticas no campo, então há aspectos importantes a serem considerados em um telefone que podem melhorar sua experiência com o CoMapeo ao longo de seu trabalho.

---

## Equipamento para CoMapeo Mobile

O CoMapeo Mobile está atualmente construído para smartphones ![android](assets/182d654c7a9e6c2b35788275066f1447fff4ca24cd0f70a34f4b06e0c4326154.png) **Android**.  Até o momento, não funciona com iPhones.

**Recursos a serem considerados ao selecionar um dispositivo**

- Duração da Bateria

- Memória

- Robustez

- Classificação à prova d'água (ou  uma boa capa de proteção)

- Qualidade da Câmera

- Sensibilidade do GPS

### Equipamento para Recursos de Colaboração do CoMapeo

O CoMapeo Mobile usa conexões Wi-Fi para compartilhar informações do projeto entre dispositivos, sem a necessidade de conexão com a internet.

Os recursos que exigem uma conexão Wi-Fi, são aqueles para colaborar com uma equipe ou adicionar um dispositivo de backup:

- Convidar novos colaboradores para um projeto

- Troca de Observações, Trilhas e Informações de Projeto

- Enviar notificação para um dispositivo que foi removido de um projeto

**Há três opções para escolher:**

![right-arrow-black](assets/241f861b8bb8e87777c4520b6df55e46a3877c02857f4e57925486e21a8e5734.png)**Roteador convencional **instalado em um escritório ou centro de comunicações

![right-arrow-black](assets/241f861b8bb8e87777c4520b6df55e46a3877c02857f4e57925486e21a8e5734.png)**Roteador portátil**

![right-arrow-black](assets/241f861b8bb8e87777c4520b6df55e46a3877c02857f4e57925486e21a8e5734.png)**Smartphone adicional **capaz de criar uma rede local offline ou hotspot.
