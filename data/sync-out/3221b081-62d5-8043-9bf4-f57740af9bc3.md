# Removing a Device from a project

Removing a device can only be done by coordinator devices. This action updates project information about the team, which is visible to all in the project.

## Coordinator Removes a Device

:::note 👣
### Step by Step

***Step 1: ***In the ![three-line-menu-black](assets/351bd29be717f58fba4061f4d5ef0db5508de31bd77417f31c2248b9f6cbbdee.jpg)**Menu**, select ![app-icon-comapeo-participant](assets/2dc5f8d1ce7cb4fc3056d93376d473bed776c782a82ce25d64e4cf658571a316.png) Team.

![image](assets/cff33ebb0b33fb38e7395b15428523442e2b4417f118a205326fc2224aaf35c4.jpg)

---

***Step 2:***  Select** **the device to be removed

![image](assets/a394086b346c5dcdb71e863f327ac37eda72b70808ee408678dd3b92c293cd83.jpg)

---

***Step 3:*** Select ![app-icon-comapeo-remove-device](assets/e688360e026ebdb5cd1019b5b7bd35136ea717e5c9b8e6af6b120df10cc89c28.jpg) **Remove device**

![image](assets/2b265bbd99ae8c4d2567b1f80b21f791a3d577fc49f4ce6be84a18edc2014e5e.jpg)

---

***Step 4:*** Write the reason for removal. This will be visible to the device being removed.

![image](assets/1e8e45095d2e96bdc33dbe6ed2aecbcd5a541bb972d5fa62dda84e3c97d0c73b.jpg)

---

***Step 5:*** Close the Keyboard and select ![app-icon-comapeo-remove-device](assets/e688360e026ebdb5cd1019b5b7bd35136ea717e5c9b8e6af6b120df10cc89c28.jpg) **Remove and notify**

![image](assets/d77b2ef0bfe85e2b83e7f7c11aa09917bc103c24b4ec9626aef7ff5db266c20d.jpg)

---

***Step 6:*** In this coordinator device, the selected device has been removed. 

![image](assets/1feeccfc2279dfa6ba616102b705b08a20d07a8ffa9ce8920368b74f0acd3245.jpg)

:::note ⚠️ Warning
The devices removed need to be connected to the same Wi-Fi router or have Remote Archive set up in order to receive the removal notification. If this does not happen, the removed devices  will continue in the project until the next time they connect with a different device in the project that has updated project information .
:::

---

***Step 6:*** Devices that were previously in a project are listed at the bottom.

![image](assets/984f67db53a2de47de665f3469ddeed394cec69cb6d8d6b46374f7735a3a5ae5.jpg)
:::

<div class="notion-spacer" aria-hidden="true" role="presentation"></div>

### Notifying the Removed Device

![image](assets/3da284d009c2231e0a62ede7e0ca3355481a6696f5dd9479a456a52b68c0c884.jpg)

In an offline context, the notification of removal cannot be sent until the devices have connected to the same Wi-Fi router at the same time. The notification will include the reason written on the coordinator device that removed them.

### Updating project information from devices in the team

![image](assets/a6c9aae8edbb73afad3751cf620111f5226a30aac381e69a4d712bb8795c989f.jpg)

The ![app-icon-comapeo-participant](assets/2dc5f8d1ce7cb4fc3056d93376d473bed776c782a82ce25d64e4cf658571a316.png) **Team** list is updated every time devices ![app-icon-comapeo-exchange](assets/d1f75c78a51b8e5a813245aa74758646c26969350099762a8dc34b327ee517c7.png) **Exchange. **

:::note 💡 Tip
The notification about device removal can be passed on through any device that has exchanged the updated information about the removed device.
:::
